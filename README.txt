========================================
HOW TO ADD YOUR FILES (ONE TIME ONLY)
========================================

STEP 1 — Profile Photo:
  Copy your photo into the portfolio/ root folder
  Rename it exactly to:  photo.jpg
  (supports .jpg, .jpeg, .png — update src in index.html if needed)

STEP 2 — Resume:
  Copy your resume PDF into the portfolio/ root folder
  Rename it exactly to:  resume.pdf

STEP 3 — Certificates (place inside this certs/ folder):
  btech-cert.jpg      → B.Tech certificate
  inter-cert.jpg      → Intermediate certificate
  ssc-cert.jpg        → SSC / Class X certificate
  html-cert.jpg       → HTML5 skill certificate
  css-cert.jpg        → CSS3 skill certificate
  js-cert.jpg         → JavaScript skill certificate
  c-cert.jpg          → C Programming certificate
  python-cert.jpg     → Python certificate
  hackathon1.jpg      → Hackathon certificate
  workshop-cert.jpg   → Workshop certificate
  academic-cert.jpg   → Academic activity certificate
  extra-cert.jpg      → Other certification

FINAL FOLDER STRUCTURE:
  portfolio/
  ├── index.html
  ├── style.css
  ├── script.js
  ├── photo.jpg          ← YOUR PHOTO HERE
  ├── resume.pdf         ← YOUR RESUME HERE
  └── certs/
      ├── btech-cert.jpg
      ├── html-cert.jpg
      └── ... (all your certificate images)

Once files are placed, open index.html in browser — everything loads automatically!
No upload buttons, no prompts. Just works.
